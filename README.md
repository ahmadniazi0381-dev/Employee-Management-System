# Employee Management System (EMS)

Employee Management System (EMS) is a desktop-based Human Resource Management application developed using Java Swing, JDBC, and MySQL. The system provides a centralized platform for managing employee records, departments, authentication, reporting, and organizational data through an intuitive graphical user interface.

The application follows Object-Oriented Programming principles and a layered architecture consisting of Model, DAO, Utility, and UI components. It enables HR administrators to perform employee CRUD operations, manage departments, generate reports, monitor organizational statistics, and maintain data integrity through validation and secure authentication.

### Key Features

* Secure user authentication and session management
* Employee record management (Create, Read, Update, Delete)
* Department management with relational database support
* Advanced employee search and filtering
* Dashboard with live organizational statistics
* Department-wise and salary-range reports
* Input validation and exception handling
* MySQL database integration using JDBC
* Modular and maintainable OOP-based architecture

### Technologies

* Java (JDK 17/21)
* Java Swing
* MySQL 8.x
* JDBC
* Maven
* Git & GitHub

This project was developed as a semester project for Software Construction & Development to demonstrate practical implementation of Object-Oriented Programming, GUI development, database management, and software engineering principles.
